## 2. Explore Tauri Security Capabilities (For BlkSpace)

Tauri is **secure by design**. Here are the critical security features you should leverage:

| Feature | What it does | How to use for BlkSpace |
| --- | --- | --- |
| **Capabilities System** | Whitelist which JS APIs can call which Rust commands. | Define `capabilities/main.json` – only allow `wallet_connect`, `upload_media`, `get_balance`. |
| **IPC Scope** | Restrict allowed origins and arguments. | Set `"windows": ["main"]` and validate all arguments in Rust. |
| **Asset Protocol** | Prevent arbitrary file reads. | Use `asset://` protocol only for whitelisted directories (e.g., `$ASSETS/`). |
| **Secure Randomness** | Built-in CSPRNG for wallet nonces. | Use `tauri::generate_random()` for transaction IDs. |
| **Context Isolation** | Isolate your frontend from the Tauri API bridge. | Enabled by default – ensures malicious JS can't spoof commands. |
| **CSP (Content Security Policy)** | Prevent XSS and script injection. | Add to `tauri.conf.json` – restrict `script-src 'self'`. |

**Example `capabilities/main.json`** (minimal but safe):

```json
{
  "identifier": "main-cap",
  "description": "Core permissions for BlkSpace",
  "windows": ["main"],
  "permissions": [
    "core:default",
    "core:window:allow-close",
    "core:window:allow-hide",
    "dialog:allow-open",
    "fs:allow-read",
    "fs:allow-write",
    {
      "identifier": "core:command:allow",
      "allow": [
        "transfer_tokens",
        "upload_media",
        "get_balance"
      ]
    }
  ]
}
```
